# JuriMesh App

## Start de app

### Database
1. Maak een PostgreSQL database aan
2. Kopieer `.env.example` naar `.env` in de backend map
3. Vul je database gegevens in

### Backend starten
```bash
cd backend
npm install
npm start
```

### Frontend starten
```bash
cd frontend
npm install
npm run dev
```

Frontend draait op: http://localhost:5173
Backend draait op: http://localhost:3000
